# 🍕 PizzaFlow - Mini System Zamówień Pizzerii

**PizzaFlow** to przykładowa aplikacja do obsługi zamówień pizzerii, stworzona w Symfony (PHP 8.2) i Dockerze.

## 🚀 Funkcjonalności

- Tworzenie i zarządzanie zamówieniami pizzy  
- Obsługa różnych rodzajów pizzy i dodatków  
- Panel administracyjny dla restauracji  
- Integracja z bazą danych MySQL / PostgreSQL  

## Dostęp do aplikacji

- Frontend (klient): http://localhost:8080
- Admin Panel: http://localhost:8080/admin/orders
